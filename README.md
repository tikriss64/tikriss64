## [AI Inbox Assistant](https://github.com/tikriss64/ai-inbox-assistant)

Self-hosted AI-powered Gmail inbox. Reads, categorizes and summarizes emails automatically via a cron job. Built from zero to production.

Gemini Flash (multimodal) for classification, Groq as backup. Features: Focus mode, Tone Composer, Snooze, Document extractor, Risks and Waiting tabs, Agenda extracted from emails. Privacy-first — runs entirely on the user's own Cloudflare account.

`React` `TanStack Start` `TypeScript` `Cloudflare Workers` `D1` `Vectorize` `Gmail OAuth` `Gemini` `Groq`

## [FUSION AI Platform](https://github.com/tikriss64/fusion-ai-platform)

Business management monorepo unifying CRM, AI Inbox, Field Operations Agenda and Web lead capture. Hybrid AI architecture: Gemini Flash for navigation and voice input, Groq for complex tasks, local command router handling ~80% of commands with zero tokens and instant response. Row-Level Security via Supabase RLS. AI assistant accessible by voice and text for creating clients, quotes and invoices in natural language.

`React` `TanStack Start` `Turborepo` `Supabase` `PostgreSQL` `pgvector` `Cloudflare Workers` `Gemini` `Groq` `Biome`

## [Valoria](https://github.com/tikriss64/valoria)

SaaS for automatic object valuation from photos. Users upload photos; OpenAI Vision analyzes them and delivers a complete HTML report by email: description, price estimate, ready-to-copy sales listing text, and recommended platforms. Includes an internal real-time supervision dashboard monitoring all services.

`React` `Node.js` `Railway` `Supabase` `OpenAI Vision` `Resend`
